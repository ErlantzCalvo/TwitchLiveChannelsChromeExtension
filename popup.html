First Time Setup

If you're building this yourself, you need to:

  1. Go to Twitch Console
  2. Create a new application
  3. Add OAuth Redirect URL: https://your-extension-id.chromiumapp.org/
  4. Copy the Client ID and paste it in the code (CLIENT_ID constant)
